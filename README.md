 Ele apresenta cards de filmes, botão de favoritar, modo escuro e responsividade básica.<br>

## 📌 Funcionalidades Principais
✅ 1. Cards de Filmes Cada card contém:
Imagem do pôster
Título do filme
Gênero
Botão “Favoritar”

✅ 2. Efeito de Hover
Ao passar o mouse sobre qualquer card: O card aumenta de tamanho
Uma sombra suave é aplicada
Transição suave via transition

✅ 3. Botão “Favoritar”
Ao clicar: O card recebe borda dourada
Ganha uma sombra especial
Pode desfavoritar ao clicar novamente
Funciona usando .classList.toggle()

✅ 4. Modo Escuro / Claro
Botão no topo permite alternar entre os modos: Fundo escuro
Cartões escuros Texto claro
Botão muda entre “Modo Escuro” e “Modo Claro”

✅ 5. Logo Centralizado Um logo é exibido no topo da página, acima do título.
